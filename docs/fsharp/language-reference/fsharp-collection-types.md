이 항목을 검토 하 여 확인할 수 있습니다 F# 컬렉션 형식 최상의 특정 요구에 적합 합니다. 이러한 컬렉션 형식에서 같은.NET framework에서 컬렉션 형식에서 다른 합니다 `System.Collections.Generic` 네임 스페이스에는 F# 개체 지향 보다는 기능을 프로그래밍 관점에서 설계 된 컬렉션 형식 큐브 뷰입니다. 보다 구체적으로, 배열 컬렉션만 요소가 변경할 수 있습니다. 따라서 컬렉션을 수정 하면 원래 컬렉션을 변경 하는 대신 수정 된 컬렉션의 인스턴스를 만듭니다.
