# <a name="displaying-data-in-the-windows-forms-datagridview-control"></a>Windows Forms DataGridView 컨트롤에서 데이터 표시
`DataGridView` 컨트롤은 다양 한 외부 데이터 원본에서에서 데이터를 표시 하는 데 사용 합니다. 또는 컨트롤에 행 및 열을 추가 하 고 수동으로 데이터로 채울 수 있습니다.  
  
 컨트롤을 데이터 소스에 바인딩하는 경우에 데이터 원본의 스키마에 따라 자동으로 열을 생성할 수 있습니다. 하고자 하는 것 처럼 이러한 열 표시 되지 않으면, 숨길 수 있습니다, 그리고 제거 또는 다시 정렬할 합니다. 또한 데이터 원본에서 제공 되지 않는 추가 데이터를 표시할 바인딩되지 않은 열을 추가할 수 있습니다.  
  
 하지만 또한 표준 형식 (예: 통화 형식)를 사용 하 여 데이터를 표시 하거나 표시 (예: 문자열 값을 바꾸거나 음수에 대 한 배경색을 변경 해야 데이터를 표시 해야 형식을 사용자 지정할 수 있습니다. 해당 이미지).
