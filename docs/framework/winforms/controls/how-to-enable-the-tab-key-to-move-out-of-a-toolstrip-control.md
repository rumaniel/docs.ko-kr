# <a name="how-to-enable-the-tab-key-to-move-out-of-a-toolstrip-control"></a>방법: TAB 키를 사용하여 ToolStrip 컨트롤 밖으로 이동하도록 설정
사용자의 이동 하려면 TAB 키를 사용 하도록 설정 하려면 다음 절차는 <xref:System.Windows.Forms.ToolStrip> 탭 순서의 다음 컨트롤로 합니다.  
  
 <xref:System.Windows.Forms.ToolStrip> TAB 키와 화살표 키 선택 항목의 첫 번째 눌러 허용 된 <xref:System.Windows.Forms.ToolStrip>합니다. 사용자가 TAB 키를 두 번 누를 때 탭 순서에서 다음 컨트롤에 사용자를 사용 합니다.  
  
### <a name="to-enable-the-user-to-press-the-tab-key-to-move-out-of-a-toolstrip-to-the-next-control"></a>다음 컨트롤로 ToolStrip 밖으로 이동 하려면 TAB 키를 눌러 사용자를 사용 하도록 설정 하려면  
  
- 설정 합니다 <xref:System.Windows.Forms.ToolStrip.TabStop%2A> 의 속성을 <xref:System.Windows.Forms.ToolStrip> 에 `true`.
