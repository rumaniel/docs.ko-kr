# <a name="imetadataemitdeletetoken-method"></a>IMetaDataEmit::DeleteToken 메서드
현재 메타 데이터 범위에서 지정된 된 토큰을 삭제합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT DeleteToken (   
    [in]  mdToken     tkObj   
);  
```  
  
## <a name="parameters"></a>매개 변수  
 `tkObj`  
 [in] 삭제할 토큰입니다.
