# <a name="imetadatafiltermarktoken-method"></a>IMetaDataFilter::MarkToken 메서드
지정 된 메타 데이터 토큰을 처리 했는지를 나타내는 값을 설정 합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT MarkToken (  
    [in] mdToken   tk  
);  
```  
  
## <a name="parameters"></a>매개 변수  
 `tk`  
 [in] 토큰 처리 됨으로입니다.
