# <a name="imetadataemitdefineuserstring-method"></a>IMetaDataEmit::DefineUserString 메서드
지정된 된 리터럴 문자열에 대 한 메타 데이터를 토큰을 가져옵니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT DefineUserString (   
    [in]  LPCWSTR     szString,   
    [in]  ULONG       cchString,   
    [out] mdString    *pstk   
);  
```  
  
## <a name="parameters"></a>매개 변수  
 `szString`  
 [in] 사용자 문자열 저장소입니다.  
  
 `cchString`  
 [in] 와이드 문자 수가 `szString`합니다.  
  
 `pstk`  
 [out] 할당 된 문자열 토큰입니다.
