# <a name="cleardownloadcache-function"></a>ClearDownloadCache 함수
다운로드 한 어셈블리의 전역 어셈블리 캐시를 지웁니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT ClearDownloadCache ();  
```  
  
## <a name="requirements"></a>요구 사항  
 **플랫폼** [시스템 요구 사항](../../get-started/system-requirements.md)을 참조하십시오.  
  
 **헤더:** Fusion. h  
  
 **라이브러리** Fusion 및 Mscorwks.dll. Mscorwks.dll 대신 Fusion을 사용 하 여 올바른 버전의 .NET Framework를 대상으로 하는지 확인 합니다.
