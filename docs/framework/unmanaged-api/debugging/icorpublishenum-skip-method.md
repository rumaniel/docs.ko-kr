# <a name="icorpublishenumskip-method"></a>ICorPublishEnum::Skip 메서드
열거형에서 항목의 지정된 된 수 만큼 앞으로 커서를 이동합니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT Skip (  
    [in] ULONG   celt  
);  
```  
  
## <a name="parameters"></a>매개 변수  
 `celt`  
 [in] 커서를 앞으로 이동에 사용 되는 항목의 수입니다.
