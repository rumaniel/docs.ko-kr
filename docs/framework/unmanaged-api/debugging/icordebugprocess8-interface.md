# <a name="icordebugprocess8-interface"></a>ICorDebugProcess8 인터페이스
[.NET Framework 4.6 및 이상 버전에서 지원 됨]  
  
 특정 유형의 사용할지 ICorDebugProcess 인터페이스를 논리적으로 확장 [ICorDebugManagedCallback2](../../../../docs/framework/unmanaged-api/debugging/icordebugmanagedcallback2-interface.md) 예외 콜백을 합니다.  
  
## <a name="methods"></a>메서드  
  
|메서드|설명|  
|------------|-----------------|  
|[EnableExceptionCallbacksOutsideOfMyCode 메서드](../../../../docs/framework/unmanaged-api/debugging/icordebugprocess8-enableexceptioncallbacksoutsideofmycode-method.md)|특정 유형의 사용할지 [ICorDebugManagedCallback2](../../../../docs/framework/unmanaged-api/debugging/icordebugmanagedcallback2-interface.md) 예외 콜백을 합니다.|
