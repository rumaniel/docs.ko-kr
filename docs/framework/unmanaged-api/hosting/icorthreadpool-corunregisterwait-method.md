이 메서드는 .NET Framework 인프라를 지원하며 사용자 코드에서 직접 사용할 수 없습니다.  
  
## <a name="syntax"></a>구문  
  
```cpp  
HRESULT CorUnregisterWait (  
    [in] HANDLE hWaitObject,

    [out] BOOL* result
```  
