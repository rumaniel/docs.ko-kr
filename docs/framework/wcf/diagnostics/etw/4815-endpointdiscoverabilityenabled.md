 이 이벤트는 엔드포인트의 검색 기능을 사용하도록 설정하면 내보내집니다.

 EndpointAddress='%1' 및 ListenUri='%2'을(를) 사용하는 엔드포인트의 검색 기능이 사용하도록 설정되어 있습니다.
