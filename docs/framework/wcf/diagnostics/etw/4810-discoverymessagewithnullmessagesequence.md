## <a name="description"></a>설명  
 이 이벤트는 DiscoveryMessageSequence 속성이 없어 검색 메시지가 DiscoveryClient에서 삭제될 때 내보내집니다.  
  
## <a name="message"></a>메시지  
 DiscoveryMessageSequence 속성이 없어 messageId='%2'인 %1 메시지가 DiscoveryClient에서 삭제되었습니다.
