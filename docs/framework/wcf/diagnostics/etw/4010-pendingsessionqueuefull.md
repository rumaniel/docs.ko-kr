|||  
|-|-|  
|ID|4010|  
|키워드|ActivationServices|  
|수준|경고|

## <a name="description"></a>설명  
 이 이벤트는 보류 중인 세션 큐가 꽉 차서 세션 디스패치가 실패한 경우에 내보내집니다.  
  
## <a name="message"></a>메시지  
 보류 중인 항목이 '%2'개로 보류 중인 세션 큐가 꽉 차서 '%1'에 대한 세션을 디스패치하지 못했습니다.
