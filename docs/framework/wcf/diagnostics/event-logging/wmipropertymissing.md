# <a name="wmipropertymissing"></a>WmiPropertyMissing
ID: 129  
  
 심각도: 오류  
  
 범주 ServiceModel  
  
## <a name="description"></a>Description  
 이 이벤트는 유형 불일치가 발생했음을 나타냅니다. 지정된 CIM 클래스에는 특정 유형의 지정된 속성이 없습니다. 이 이벤트는 프로세스 이름과 프로세스 ID를 표시합니다.
