# <a name="custom-tokens"></a>사용자 지정 토큰
Windows Communication Foundation (WCF)는 기본적으로 x.509 인증서, 보안 컨텍스트 토큰, Kerberos 티켓 및 사용자 이름 토큰을 자격 증명으로 지원 하지만 고유한 사용자 지정 자격 증명을 사용할 수 있을 만큼 유연 합니다.  
  
## <a name="in-this-section"></a>섹션 내용  
 [방법: 사용자 지정 토큰 만들기](how-to-create-a-custom-token.md)  
 <xref:System.IdentityModel.Tokens.SecurityToken> 클래스를 사용하여 사용자 지정 보안 토큰을 만들고 사용자 지정 보안 토큰 공급자 및 인증자를 사용하여 통합하는 방법에 대해 설명합니다.
