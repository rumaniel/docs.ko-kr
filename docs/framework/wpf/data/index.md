# <a name="data"></a>보기
[!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] 데이터 바인딩은 응용 프로그램이 데이터를 제공하고 상호 작용할 수 있는 간단하고 일관된 방법을 제공합니다. 요소는 CLR (공용 언어 런타임) 개체 및 XML 형식으로 다양 한 데이터 소스의 데이터에 바인딩될 수 있습니다. [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)] 또한 끌어서 놓기 작업을 통해 데이터 전송을 위한 메커니즘을 제공합니다.
