|-|-|  
|ID|1012|  
|키워드|WFRuntime|  
|수준|자세히|  
|채널|Microsoft-Windows-애플리케이션 서버-애플리케이션/디버그|

 ExecuteActivityWorkItem이 실행을 시작했음을 나타냅니다.
