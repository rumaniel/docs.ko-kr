# <a name="c-special-characters"></a>C# 특수 문자

특수 문자는 해당 문자가 앞에 붙는 프로그램 요소(리터럴 문자열, 식별자 또는 특성 이름)를 수정하는 미리 정의된 상황에 맞는 문자입니다. C#은 다음과 같은 특수 문자를 지원합니다. 

- [@](./verbatim.md), 약어 식별자 문자입니다. 

- [$](./interpolated.md), 보간된 문자열 문자입니다.
