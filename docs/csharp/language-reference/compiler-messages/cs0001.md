# <a name="compiler-error-cs0001"></a>컴파일러 오류 CS0001
내부 컴파일러 오류

 컴파일러가 예기치 않은 구문을 분석할 수 없어 실패하는지 여부를 확인합니다. 이 오류가 반복해서 발생하면 Microsoft에 문의하세요.
