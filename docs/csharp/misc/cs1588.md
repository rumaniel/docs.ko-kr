# <a name="compiler-error-cs1588"></a>컴파일러 오류 CS1588
공용 언어 런타임 디렉터리를 확인할 수 없습니다. 'reason'  
  
 컴파일러가 .NET Framework 공용 언어 런타임이 설치된 디렉터리를 검색할 수 없습니다. 이 오류는 공용 언어 런타임이 잘못 설치되었음을 나타냅니다.
