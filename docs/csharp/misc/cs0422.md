# <a name="compiler-warning-level-4-cs0422"></a>컴파일러 경고(수준 4) CS0422

/incremental 옵션은 더 이상 지원되지 않습니다.

 증분 컴파일 ( **/incr** 또는 **/incremental**)은 2.0에서 C# 지원 되지 않습니다.
 
> [!NOTE]
> 이 컴파일러 경고는 Roslyn에서 더 이상 사용 되지 않습니다.
