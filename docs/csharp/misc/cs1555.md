# <a name="compiler-error-cs1555"></a>컴파일러 오류 CS1555
Main 메서드에 지정된 'class'를 찾을 수 없습니다.  
  
 [/main](../language-reference/compiler-options/main-compiler-option.md) 컴파일러 옵션에 대해 클래스가 지정되었지만 소스 코드에 클래스 이름이 없습니다.
