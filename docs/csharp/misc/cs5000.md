# <a name="compiler-warning-level-1-cs5000"></a>컴파일러 경고(수준 1) CS5000

알 수 없는 컴파일러 옵션 '/option'입니다.

 잘못된 [컴파일러 옵션](../language-reference/compiler-options/index.md) 이 지정되었습니다.

> [!NOTE]
> 이 컴파일러 오류는 Roslyn에서 더 이상 사용 되지 않습니다.
