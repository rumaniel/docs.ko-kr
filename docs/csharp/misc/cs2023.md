# <a name="compiler-warning-level-1-cs2023"></a>컴파일러 경고(수준 1) CS2023
지시 파일에 지정되었기 때문에 /noconfig 옵션을 무시합니다.  
  
 [/noconfig](../language-reference/compiler-options/noconfig-compiler-option.md) 컴파일러 옵션이 지시 파일에서 지정되었으며 이는 허용되지 않습니다.  
  
 이 경고는 [/nowarn](../language-reference/compiler-options/nowarn-compiler-option.md) 옵션으로 표시하지 않을 수 없습니다.
