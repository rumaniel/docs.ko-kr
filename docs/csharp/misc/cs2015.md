# <a name="compiler-error-cs2015"></a>컴파일러 오류 CS2015
'file'은 텍스트 파일이 아니라 이진 파일입니다.  
  
 파일이 이진 파일인 컴파일러에 전달되었습니다. 컴파일러에 소스 코드 파일이 필요합니다.
