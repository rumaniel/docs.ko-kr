# <a name="compiler-error-cs1556"></a>컴파일러 오류 CS1556
Main 메서드에 지정된 'construct'는 유효한 클래스나 구조체여야 합니다.  
  
 [/main](../language-reference/compiler-options/main-compiler-option.md) 컴파일러 옵션에 클래스 이름이 아닌 식별자가 전달되었습니다.
