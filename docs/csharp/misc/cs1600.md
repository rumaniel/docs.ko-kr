# <a name="compiler-error-cs1600"></a>컴파일러 오류 CS1600
사용자가 컴파일을 취소했습니다.  
  
 Visual Studio IDE를 사용하는 동안 C# 컴파일러를 사용한 컴파일이 취소되었습니다.
