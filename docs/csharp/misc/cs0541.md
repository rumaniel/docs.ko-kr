# <a name="compiler-error-cs0541"></a>컴파일러 오류 CS0541
'declaration': 명시적 인터페이스 선언은 클래스나 구조체에서만 선언할 수 있습니다.  
  
 명시적 [인터페이스](../language-reference/keywords/interface.md) 선언을 [클래스](../language-reference/keywords/class.md) 또는 [구조체](../language-reference/keywords/struct.md)외부에서 찾았습니다.  
  
 다음 샘플에서는 CS0541을 생성합니다.  
  
```csharp  
// CS0541.cs  
namespace x  
{  
   interface IFace  
   {  
      void F();  
   }  
  
   interface IFace2 : IFace  
   {  
      void IFace.F();   // CS0541  
   }  
}  
```
