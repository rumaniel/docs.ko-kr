# <a name="compiler-error-cs1664"></a>컴파일러 오류 CS1664
길이가 'length'인 'type' 형식의 고정 크기 버퍼가 너무 큽니다.  
  
 길이에 요소 크기를 곱한 값으로 결정되는 고정 크기 버퍼의 최대 크기는 2^31 = 268435455입니다.
