# <a name="compiler-error-cs1606"></a>컴파일러 오류 CS1606
어셈블리에 서명하지 못했습니다. 출력이 서명되지 않은 상태일 수 있습니다. reason  
  
 어셈블리가 생성되었지만 컴파일러가 서명을 완료하려고 할 때 오류가 발생했습니다.
