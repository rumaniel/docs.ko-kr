# <a name="compiler-error-cs0517"></a>컴파일러 오류 CS0517
'class'에는 기본 클래스가 없으므로 기본 생성자를 호출할 수 없습니다.  
  
 CS0517은 .NET Framework 공용 언어 런타임이 기본 클래스가 없는 유일한 클래스인 개체 클래스에 대한 소스 코드를 컴파일하는 경우에만 발생할 수 있습니다.
