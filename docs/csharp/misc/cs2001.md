# <a name="compiler-error-cs2001"></a>컴파일러 오류 CS2001
'file' 원본 파일을 찾을 수 없습니다.  
  
 원본 파일 이름이 컴파일러로 전달되었지만 찾을 수 없습니다. 파일 이름과 파일 위치의 철자를 확인합니다.
