# <a name="compiler-error-cs1541"></a>컴파일러 오류 CS1541
잘못된 참조 옵션: 'symbol' -- 디렉터리를 참조할 수 없습니다.  
  
 컴파일러가 특정 파일이 아닌 디렉터리를 지정하려는 시도를 검색했습니다. 예를 들어 [/reference](../language-reference/compiler-options/reference-compiler-option.md) 컴파일러 옵션을 사용하는 경우 파일을 지정해야 하며 디렉터리를 지정할 수 없습니다.  
  
 예를 들어 컴파일러에 `/reference:c:\` 를 전달하면 CS1541이 생성됩니다.
