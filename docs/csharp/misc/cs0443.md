# <a name="compiler-error-cs0443"></a>컴파일러 오류 CS0443
구문 오류입니다. 값이 필요합니다.  
  
 이 오류는 배열 인덱스에 대한 값을 지정하지 않고 배열을 참조하는 경우 발생합니다.  
  
## <a name="example"></a>예제  
 다음 코드에서는 CS0443을 생성합니다.  
  
```csharp  
// CS0443.cs   
using System;   
class MyClass   
{  
    public static void Main()      
    {  
        int[,] x = new int[1,5];  
        if (x[] == 5) {} // CS0443  
        // if (x[0, 0] == 5) {}   
    }  
}  
```
