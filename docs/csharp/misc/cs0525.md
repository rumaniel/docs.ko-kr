# <a name="compiler-error-cs0525"></a>컴파일러 오류 CS0525
인터페이스는 필드를 포함할 수 없습니다.  
  
 [interface](../language-reference/keywords/interface.md) 에 메서드 및 속성이 포함될 수 있지만 필드는 포함되지 않습니다.  
  
 다음 샘플에서는 CS0525를 생성합니다.  
  
```csharp  
// CS0525.cs  
namespace x  
{  
   public interface clx  
   {  
      public int i;   // CS0525  
   }  
}  
```
