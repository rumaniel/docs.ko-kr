# <a name="property-not-found"></a>속성을 찾을 수 없습니다.
이 개체는 지정된 된 속성을 지원 하지 않습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. 속성의 이름의 철자를 확인 합니다.  
  
2. "Caption" 또는 이름이 비슷한 속성을 개체를 실제로 지 원하는 경우 "text" 속성 처럼 항목에 액세스를 시도 하지 않는 되도록 개체의 설명서를 확인 합니다.
