# <a name="optional-expected"></a>'Optional'이 필요합니다.
프로시저 선언에서 선택적 인수는 필수 인수가 옵니다. 모든 인수는 선택적 인수 뒤에 선택적 수도 있어야 합니다.  
  
 **오류 ID:** BC30202  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. 인수는 필요할 경우에 인수 목록의 첫 번째 선택적 인수 앞으로 이동 합니다.  
  
2. 선택적 인수는 사용 하는 경우 사용 된 `Optional` 키워드입니다.
