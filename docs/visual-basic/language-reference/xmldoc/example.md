# <a name="example-visual-basic"></a>\<예제 > (Visual Basic)
멤버에 대 한 예제를 지정합니다.  
  
## <a name="syntax"></a>구문  
  
```xml  
<example>description</example>  
```  
  
## <a name="parameters"></a>매개 변수  
 `description`  
 코드 샘플에 대한 설명입니다.  
  
## <a name="remarks"></a>설명  
 `<example>` 태그는 메서드 또는 기타 라이브러리 멤버를 사용 하는 방법의 예제를 지정할 수 있습니다. 여기에는 일반적으로 [\<code>](../../../visual-basic/language-reference/xmldoc/code.md) 태그를 같이 사용합니다.  
  
 [/doc](../../../visual-basic/reference/command-line-compiler/doc.md)로 컴파일하여 문서 주석을 파일로 처리합니다.
