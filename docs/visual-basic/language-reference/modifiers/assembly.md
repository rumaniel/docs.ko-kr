# <a name="assembly-visual-basic"></a>Assembly(Visual Basic)
소스 파일의 시작 부분에 있는 특성이 전체 어셈블리에 적용 되도록 지정 합니다.  
  
## <a name="remarks"></a>설명  
 많은 특성이 클래스나 속성 같은 개별 프로그래밍 요소에 관련이 있습니다. 꺾쇠 괄호를 특성 블록을 연결 하 여 특성을 적용 하면 (`< >`), 선언문에 직접.  
  
 특성에는 다음 요소 뿐 아니라 전체 어셈블리에 관련 된, 소스 파일의 시작 부분에 특성 블록을 배치를 사용 하 여 특성을 식별 합니다 `Assembly` 키워드입니다. 현재 어셈블리 모듈에 해당 되는 경우 사용 합니다 [모듈](../../../visual-basic/language-reference/modifiers/module-keyword.md) 키워드입니다.  
  
 적용할 수 있습니다도 특성 AssemblyInfo.vb 파일에서 어셈블리에 경우 특성 블록을 사용 하 여 주 소스 코드 파일에서 필요가 없습니다.
