# <a name="cannot-convert-start-value-of-type-type1-limit-value-of-type2-and-step-value-of-type3-to-a-common-type"></a>형식의 시작 값으로 변환할 수 없습니다 '\<type1 >'의 한계 값 '\<type2 >' 형식의 단계 값 및 '\<type3 >'을 공용 형식
지정된 `For...Next` 컨트롤 값의 형식이 서로 달라서 공통 형식으로 변환할 수 없으므로 유효하지 않습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. 모두 공통 형식으로 변환될 수 있도록 하나 이상의 값 형식을 변경합니다.
