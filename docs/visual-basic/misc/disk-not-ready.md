# <a name="disk-not-ready"></a>디스크가 준비되지 않았습니다.
지정된 드라이브에 대해 디스크가 없거나 드라이브 문이 열려 있습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. 지정된 드라이브를 확인하고 필요한 경우 디스크를 삽입합니다.  
  
2. 드라이브 문을 닫습니다.
