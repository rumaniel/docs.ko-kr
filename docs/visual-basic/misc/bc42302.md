# <a name="xml-comment-must-be-the-first-statement-on-a-line"></a>XML 주석은 줄의 첫째 문이어야 함
XML 주석은 줄의 첫째 문이어야 합니다. XML 주석이 무시됩니다.  
  
 XML 주석이 다른 언어 요소 뒤에 있는데 이는 허용되지 않습니다.  
  
 **오류 ID:** BC42302  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- XML 주석을 새 줄로 이동합니다.
