# <a name="property-missing-end-property"></a>속성에 'End Property'가 없습니다.
`End` 문은 종료하는 블록과 일치해야 합니다. `Property` 정의가 제대로 종료되지 않았습니다.  
  
 **오류 ID:** BC30025  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 속성 정의의 끝에 `End Property` 문을 추가합니다.
