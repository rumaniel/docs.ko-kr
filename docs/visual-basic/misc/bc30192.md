# <a name="end-of-parameter-list-expected"></a>매개 변수 목록의 끝이 필요합니다.
프로시저 선언에서 `ParamArray` 사양 뒤에 하나 이상의 다른 인수 사양이 옵니다.  
  
 **오류 ID:** BC30192  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- `ParamArray` 를 인수 목록의 끝으로 이동합니다.
