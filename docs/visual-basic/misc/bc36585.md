# <a name="elementname-is-not-a-parameter-of-extension-method-methodname-defined-in-typename"></a>'\<elementname >' 확장명 메서드의 매개 변수가 아닙니다. '\<methodname >'에 정의 된 '\<typename >'
프로시저 호출이 확장 메서드에서 정의되지 않은 추가 인수를 제공합니다.  
  
 **오류 ID:** BC36585  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 인수 목록에서 불필요한 인수를 제거합니다.
