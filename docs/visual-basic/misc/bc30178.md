# <a name="specifier-is-duplicated"></a>지정자가 중복되었습니다.
키워드가 동일한 선언에 두 번 이상 표시됩니다.  
  
 **오류 ID:** BC30178  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 선언에서 키워드의 중복 표시를 제거합니다.
