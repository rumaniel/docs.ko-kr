# <a name="typename-must-be-declared-mustinherit-because-it-contains-methods-declared-mustoverride"></a>'\<typename >' 선언 'MustOverride' 메서드를 포함 하므로 'MustInherit' 선언 해야 합니다
`MustOverride` 멤버를 포함하는 형식이 `MustInherit`로 선언되어야 합니다.  
  
 **오류 ID:** BC31411  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- `MustInherit` 한정자를 형식에 추가합니다.
