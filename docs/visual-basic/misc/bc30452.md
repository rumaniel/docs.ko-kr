# <a name="operator-operatorname-is-not-defined-for-types-type1-and-type2"></a>연산자 '\<operatorname >' 형식에 대해 정의 되지 않은\<type1 >' 및 '\<type2 >'
이항 연산자를 사용하여 관련된 두 개의 값 반환 코드 요소에 사용할 수 없는 작업을 수행하려고 했습니다.  
  
 **오류 ID:** BC30452  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 두 요소를 검사하고 호환되는지 확인합니다.
