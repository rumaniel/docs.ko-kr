# <a name="extension-method-methodname-defined-in-modulename-does-not-have-a-signature-compatible-with-delegate-delegatename"></a>확장 메서드 '\<methodname >'에 정의 된 '\<modulename >' 대리자와 호환 되는 서명 되지 않은 '\<delegatename >'
사용하려는 확장 메서드 및 대리자의 서명이 서로 일치하지 않습니다. `Delegate` 문에서 대리자 클래스의 매개 변수 형식 및 반환 형식을 정의합니다. 호환되는 매개 변수, 매개 변수 형식 및 반환 형식이 포함된 프로시저를 사용하여 이 대리자 형식의 인스턴스를 만들 수 있습니다.  
  
 **오류 ID:** BC36710
