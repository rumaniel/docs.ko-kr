# <a name="end-statements-are-not-valid-in-the-immediate-window"></a>직접 실행 창에서는 'End' 문을 사용할 수 없습니다.
`Stop` 및 `End` 문의 실행이 일시 중단되고 디버깅 컨텍스트에서 허용되지 않습니다.  
  
 **오류 ID:** BC30123  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- `End` 직접 실행 `Stop` 창에서 **또는** 문을 실행하지 않아야 합니다.
