# <a name="name-is-already-declared-as-a-parameter-of-this-method"></a>'\<이름 >'이이 메서드의 매개 변수로 이미 선언 되었습니다
지정된 매개 변수 이름이 이 메서드에 대해 이미 선언되었습니다.  
  
 **오류 ID:** BC30734  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 고유하게 매개 변수의 이름을 바꿉니다.
