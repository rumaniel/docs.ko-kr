# <a name="exit-operator-is-not-valid-use-return-to-exit-an-operator"></a>'Exit Operator'가 잘못되었습니다. 연산자를 종료하려면 'Return'을 사용하세요.
`Exit Operator` 문이 `Operator` 프로시저에 나타납니다.  
  
 사용 해야 합니다는 [Return 문이](../../visual-basic/language-reference/statements/return-statement.md) 에서 반환 하는 `Operator` 프로시저입니다. [Exit 문](../../visual-basic/language-reference/statements/exit-statement.md) 받아들이지 않는 합니다 `Operator` 키워드를 및 `End Operator` 문은 호출 코드로 컨트롤을 반환 하지 않습니다.  
  
 **오류 ID:** BC33008  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- `Exit Operator` 문을 `Return` 문으로 바꿉니다.
