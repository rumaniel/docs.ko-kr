# <a name="bracketed-identifier-is-missing-closing-"></a>대괄호로 묶은 식별자에 닫는 ']'가 없습니다.
이스케이프된 이름의 대괄호는 일치하는 쌍으로 사용해야 합니다.  
  
 **오류 ID:** BC30034  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 이스케이프된 이름의 끝에 닫는 대괄호를 추가합니다.
