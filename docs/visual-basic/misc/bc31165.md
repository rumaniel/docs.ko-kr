# <a name="expected-beginning--for-an-xml-tag"></a>필요한 시작 '\<' XML 태그
XML 리터럴의 XML 태그에 필요한 시작 '<' 문자가 없습니다.  
  
 **오류 ID:** BC31165  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- XML 리터럴의 XML 태그 시작에 '<' 문자를 추가합니다.
