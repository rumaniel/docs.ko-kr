# <a name="referenced-object-objectname-has-a-value-of-nothing"></a>참조 된 개체 '\<개체 이름 >'의 값에 'Nothing'
참조되는 개체를 설정하지 않았거나 `Nothing`값으로 설정했습니다.  
  
 **오류 ID:** BC30705  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 개체 이름의 철자가 맞는지, 개체가 설정되었는지 확인합니다.
