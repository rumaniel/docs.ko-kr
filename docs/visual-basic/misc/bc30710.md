# <a name="end-of-expression-expected"></a>식의 끝이 필요합니다.
완료된 식의 끝에서 추가 문자를 발견했습니다.  
  
 **오류 ID:** BC30710  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 개체 또는 문의 구문을 확인합니다.
