# <a name="expected-closing--for-xml-entity"></a>XML 엔터티에 닫는 ';'이 필요합니다.
XML 엔터티 참조(예: `>`)에 닫는 세미콜론(;)이 없습니다.  
  
 **오류 ID:** BC31178  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- XML 엔터티 참조의 끝에 세미콜론을 포함합니다.
