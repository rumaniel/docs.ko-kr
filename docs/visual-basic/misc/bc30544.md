# <a name="method-cannot-contain-both-a-try-statement-and-an-on-error-or-resume-statement"></a>메서드에는 'Try' 문과 함께 'On Error' 또는 'Resume' 문을 사용할 수 없습니다.
`Try` 문을 `On Error` 또는 `Resume`과 결합했습니다.  
  
 **오류 ID:** BC30544  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. `On Error` 또는 `Resume`을 제거합니다.
