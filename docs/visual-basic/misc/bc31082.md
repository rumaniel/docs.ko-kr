# <a name="variablename-is-not-a-local-variable-or-parameter-and-so-cannot-be-used-as-a-catch-variable"></a>'\<variablename >' 지역 변수 또는 매개 변수를 아니며 'Catch' 변수로 사용할 수 없습니다
`Try...Catch...Finally` 문의 변수는 로컬로 선언되어야 하며 현재 프로시저의 매개 변수여야 합니다.  
  
 **오류 ID:** BC31082  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. `Try...Catch...Finally` 문의 지역 변수 또는 매개 변수를 선언합니다.
