# <a name="type-parameters-cannot-be-specified-on-this-declaration"></a>이 선언에는 형식 매개 변수를 지정할 수 없습니다.
프로그래밍 요소가 형식 매개 변수 목록을 사용하여 선언되었지만 프로그래밍 요소는 제네릭 형식이 될 수 없습니다.  
  
 제네릭이 될 수 없는 프로그래밍 요소는 속성, 연산자, 이벤트 및 생성자를 포함합니다. 형식 매개 변수 목록을 사용하여 이러한 요소를 선언하면 오류가 발생합니다.  
  
 **오류 ID:** BC32065  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- `Of` 키워드 및 형식 매개 변수를 선언에서 제거합니다.
