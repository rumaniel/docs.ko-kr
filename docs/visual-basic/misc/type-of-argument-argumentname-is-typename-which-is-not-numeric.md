# <a name="type-of-argument-argumentname-is-typename-which-is-not-numeric"></a>인수 형식 '\<argumentname >'는 '\<typename >', 숫자가 아닌
숫자 인수가 필요한 상황에서 숫자가 아닌 인수가 발생합니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 인수가 숫자인지 확인합니다.
