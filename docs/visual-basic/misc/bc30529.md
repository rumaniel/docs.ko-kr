# <a name="all-parameters-must-be-explicitly-typed-if-any-are"></a>하나라도 명시적으로 형식화된 경우 모든 매개 변수를 명시적으로 형식화해야 합니다.
함수에서 다양한 형식의 선언을 혼합했습니다.  
  
 **오류 ID:** BC30529  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 모든 매개 변수가 명시적으로 형식화되었는지 확인합니다.
