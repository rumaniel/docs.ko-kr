# <a name="the-file-filename-is-not-a-text-file"></a>파일 '\<파일 이름 >'은 텍스트 파일이 아닙니다
이진 파일이기 때문에 지정된 파일에 쓸 수 없습니다.  
  
 **오류 ID:** BC2015  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 올바른 파일 이름 및 경로를 제공했는지 확인합니다.
