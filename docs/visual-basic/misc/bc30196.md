# <a name="comma-expected"></a>쉼표가 필요합니다.
인수 목록에서 두 인수 사이에 쉼표(,)가 없습니다.  
  
 **오류 ID:** BC30196  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 모든 인수기 쉼표로 구분되어 있는지 확인합니다.
