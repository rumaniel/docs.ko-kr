# <a name="structure-statement-must-end-with-a-matching-end-structure"></a>'Structure' 문은 짝이 되는 'End Structure'로 끝나야 합니다.
구조체는 `Structure` 문으로 시작하여 `End Structure` 구문으로 끝납니다.  
  
 **오류 ID:** BC30624  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 구조체 블록이 `End Structure` 문으로 끝나는지 확인합니다.
