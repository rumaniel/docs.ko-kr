# <a name="file-information-cannot-be-queried-while-open-for-writing"></a>파일이 쓰기용으로 열려 있는 동안에는 파일 정보를 쿼리할 수 없습니다.
쓰기를 위해 현재 열려 있는 파일에 대한 정보를 가져오려고 했습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 파일을 닫고 작업을 다시 시도합니다.
