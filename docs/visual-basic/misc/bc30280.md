# <a name="enum-enumname-must-contain-at-least-one-member"></a>열거형 '\<enumname에서는 >' 멤버를 하나 이상 있어야 합니다.
빈 열거형이 발생합니다. 하나 이상의 멤버가 `Enum` 문과 `End Enum` 문 사이에 선언되어야 합니다.  
  
 **오류 ID:** BC30280  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. 열거형에 멤버 선언을 추가합니다.
