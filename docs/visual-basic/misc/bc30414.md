# <a name="value-of-type-type1-cannot-be-converted-to-type2-because-the-array-types-have-different-numbers-of-dimensions"></a>형식의 값 '\<type1 >'로 변환할 수 없습니다 '\<type2 >' 배열 형식을 차원 수가 다른 있으므로
한 배열을 차원 수가 다른 배열로 변환하려고 했습니다.  
  
 **오류 ID:** BC30414  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 차원 수가 일치하도록 하나 또는 두 배열을 변경합니다.
