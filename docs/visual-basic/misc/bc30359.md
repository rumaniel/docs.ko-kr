# <a name="default-can-be-applied-to-only-one-property-name-in-a-type"></a>하나의 속성 이름에 적용할 수 '기본값'을 \<형식 >

 **오류 ID:** BC30359  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- `Default` 선언 중 하나만 남기고 모두 제거합니다.
