# <a name="type-character-cannot-be-used-in-a-type-parameter-declaration"></a>형식 매개 변수 선언에는 형식 문자를 사용할 수 없습니다.
형식 매개 변수 선언은 식별자 형식 문자를 하나 이상 포함합니다.  
  
 제네릭 형식의 형식 매개 변수는 올바른 Visual Basic 이름이어야 합니다. 허용되는 문자에 식별자 형식 문자(`%`, `&`, `@`, `!`, `#`및 `$`)는 포함되지 않습니다. [Declared Element Names](../../visual-basic/programming-guide/language-features/declared-elements/declared-element-names.md)을 참조하세요.  
  
 **오류 ID:** BC32041  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 형식 매개 변수 선언에서 형식 식별자 문자를 제거합니다.
