# <a name="expected-closing--for-xml-processor-instruction"></a>필요한 닫는 '? >' XML 프로세서 명령에 대 한
XML 처리 명령 리터럴에 필요한 닫는 `?>` 문자가 없습니다.  
  
 **오류 ID:** BC31160  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- XML 처리 명령 리터럴의 끝에 필요한 닫는 `?>` 문자를 추가합니다.
