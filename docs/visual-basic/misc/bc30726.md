# <a name="with-contexts-and-statements-are-not-valid-in-debug-windows"></a>디버그 창에서는 'With' 컨텍스트 및 문을 사용할 수 없습니다.
`With` 문은 소스 코드에서만 사용할 수 있습니다.  
  
 **오류 ID:** BC30726  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 디버깅 코드에서 `With` 문을 제거합니다.
