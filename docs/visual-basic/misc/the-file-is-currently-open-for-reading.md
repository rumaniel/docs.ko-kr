# <a name="the-file-is-currently-open-for-reading"></a>파일이 현재 읽기용으로 열려 있습니다.
현재 읽기용으로 열려 있는 파일에 액세스하려고 했습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
- 파일을 닫고 작업을 다시 시도합니다.
