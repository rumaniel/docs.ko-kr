# <a name="argument-conversion-is-not-valid"></a>'Conversion' 인수가 잘못되었습니다.
`VbStrConv` 열거형 값이 잘못되었습니다. 0보다 작은 것 같습니다. 열거형 값은 0보다 크거나 같아야 합니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. 값이 0보다 크거나 같은지 확인합니다.
