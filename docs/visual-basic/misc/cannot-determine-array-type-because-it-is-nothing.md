# <a name="cannot-determine-array-type-because-it-is-nothing"></a>배열이 Nothing이므로 배열 형식을 확인할 수 없습니다.
`Nothing`값으로 설정되어 배열 형식을 결정할 수 없습니다.  
  
## <a name="to-correct-this-error"></a>이 오류를 해결하려면  
  
1. 배열에 대해 `Nothing` 이 아닌 값을 제공합니다.
